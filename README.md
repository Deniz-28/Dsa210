# Social Media and Mental Health Analysis

## Project Description
This project analyzes the relationship between social media usage and mental health.

## Dataset
The dataset includes:
- Age
- Gender
- Daily social media usage
- Stress level

## Methods
- Exploratory Data Analysis (EDA)
- Correlation analysis
- T-test (Gender comparison)
- ANOVA (Age comparison)

## Results
The analysis shows that social media usage has a relationship with stress levels. Differences across age groups and genders were also tested.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scipy

## How to Run
Open the notebook file (analysis.ipynb) and run all cells.
